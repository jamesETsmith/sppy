# Small-molecule Plotting in Python (SPPy)
This repo is a module for lightweight molecular plotting and visualization in Python.

## Dependencies
* NumPy
* Matplotlib

## Installation
* git clone repo
* add path to repo to your PYTHONPATH in ~/.bashrc or bash profile
* Ready to use!
